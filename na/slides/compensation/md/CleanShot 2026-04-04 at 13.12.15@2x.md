# PSM+ Compensation | Quarterly Production Bonus

![Source](../images/CleanShot 2026-04-04 at 13.12.15@2x.png)

## Summary of Changes 📋

- Scheme remain unchanged, with only adjustment in P13 modifier (apply from Q2/2026)
- Active agent requirement includes SA active

## Current Scheme

Quarterly production bonus = Bonus rate * % FYC location * P13 Modifier

| % hitting total quarterly 1mAA target | Bonus for PSM+ manage directly the location | Upper managers (Indirect PSSM+) |
|---|---|---|
| ≥ 90% & < 100% | 1.0% FYC | |
| ≥ 100% & < 110% | 2.5% FYC | 50% bonus of direct report downline |
| ≥ 110% & < 120% | 3.5% FYC | |
| ≥ 120% | 5.0% FYC | |

| % Actual P13 rolling of location | % P13 modifier |
|---|---|
| < 50% | x Actual P13 |
| ≥ 50% and < 55% | 75% |
| ≥ 55% and < 60% | 90% |
| ≥ 60% and < 65% | 100% |
| ≥ 65% and < 70% | 105% |
| ≥ 70% and < 75% | 110% |
| ≥ 75% | 115% |
| No P13 yet | 100% |

- 1mAA = min 1CC & 4mil FYC per month
- Total quarterly target= sum of monthly 1mAA target in the quarter

## Proposed Scheme

Quarterly production bonus = Bonus rate * FYC location * P13 Modifier

| % hitting total quarterly 1mAA target | Bonus for PSM+ manage directly the location | Upper managers (Indirect PSSM+) |
|---|---|---|
| ≥ 90% & < 100% | 1.0% FYC | |
| ≥ 100% & < 110% | 2.5% FYC | 50% bonus of direct report downline |
| ≥ 110% & < 120% | 3.5% FYC | |
| ≥ 120% | 5.0% FYC | |

| % Actual P13 rolling of location | % P13 modifier |
|---|---|
| < 50% | x Actual P13/2 |
| ≥ 50% and < 60% | 50% |
| ≥ 60% and < 65% | 70% |
| ≥ 65% and < 70% | 90% |
| ≥ 70% and < 75% | 100% |
| ≥ 75% and < 80% | 105% |
| ≥ 80% | 110% |
| No P13 yet | 100% |

*The row "≥ 70% and < 75%" and "100%" are highlighted in green, indicating the baseline threshold*

- 1mAA = min 1CC & 4mil FYC per month, include SA active
- Total quarterly target= sum of monthly 1mAA target in the quarter
