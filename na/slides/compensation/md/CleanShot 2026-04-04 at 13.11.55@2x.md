## Summary of Changes *📋 icon*

- Include rehire ≥ 24 months
- Adjustment in P13 modifier (apply from Q2/2026)

---

## Current Scheme

New Agent Development Bonus = (A.2) * P13 modifier

**A.2- Extra New Agent Development Bonus/ Month**

**50%** * [Total NAB & Fast track bonus] of Agent direct reporting(**)

| % Actual P13 | % P13 modifier |
|---|---|
| < 50% | x actual P13 |
| ≥ 50% and < 55% | 75% |
| ≥ 55% and < 60% | 90% |
| ≥ 60% and < 65% | 100% |
| ≥ 65% and < 70% | 105% |
| ≥ 70% | 110% |
| ≥ 75% | 115% |
| no P13 yet | 100% |

- (\*\*) New agents in direct report and qualify New Agent bonus and Fast track bonus (exclude terminated agents/ rehired agent)
- Direct report is included all FA and UM+ at level 1
