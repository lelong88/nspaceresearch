# PSM+ Compensation | Monthly Allowance

![Source](../images/CleanShot 2026-04-04 at 13.12.09@2x.png)

## Summary of Changes 📋

- Replace uniform tier-based modifiers with new modifiers proportional to %target achieve (bonus modifiers = 130% x %FYP achievement modifier) to encourage PSM+ to keep achieving higher target rather than stop at bonus threshold
- Maximum bonus is 150% fix amount by rank to still encourage >115% target achievement

---

## Current Scheme

Monthly allowance = **(1)** Monthly allowance by rank * **(2) %Bonus modifier**

| Rank | (1) Monthly allowance by rank (mil VND) |
|------|----------------------------------------|
| PSM/SM | 35 |
| PSSM/SSM | 45 |
| DRD/RD | 60 |
| SRD | 80 |
| AVP | 110 |
| SAVP | 125 |
| VP | 150 |

| % hitting FYP target | (2) % Bonus Modifier |
|----------------------|---------------------|
| < 30% | 0% |
| ≥ 30% and < 60% | 50% |
| ≥ 60% and < 80% | 80% |
| ≥ 80% and < 115% | 100% |
| ≥ 115% | 120% |

Max: 120% Fix amount

---

*→ Green arrow pointing from Current Scheme to Proposed Scheme*

## Proposed Scheme

Monthly allowance = **(1)** Monthly allowance by rank * **(2) %FYP target achievement * 130%**

| Rank | (1) Monthly allowance by rank (mil VND) |
|------|----------------------------------------|
| PSM/SM | 35 |
| PSSM/SSM | 45 |
| DRD/RD | 60 |
| SRD | 80 |
| AVP | 110 |
| SAVP | 125 |
| VP | 150 |

Max: 150% Fix amount
Min requirement: % hitting FYP target ≥ 30%

*A line chart is shown comparing the current scheme (step function with tiers) versus the proposed scheme (linear proportional curve), illustrating how the proposed scheme provides continuous incentive to achieve higher targets rather than plateauing at tier thresholds.*
