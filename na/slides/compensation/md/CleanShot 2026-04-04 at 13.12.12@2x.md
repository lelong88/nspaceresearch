# PSM+ Compensation | Monthly Production Bonus

![Source](../images/CleanShot 2026-04-04 at 13.12.12@2x.png)

**Summary of Changes** 📋

- Scheme remain unchanged, with only adjustment in P13 modifier (apply from Q2/2026)

---

## Current Scheme

Monthly Production Bonus = Bonus rate * FYC location * P13 Modifier

| % hitting monthly FYP target of sales location | Bonus for PSM+ manage directly the location | Upper managers (Indirect PSSM+) |
|---|---|---|
| < 115% | 6% FYC | 50% bonus of direct report downline |
| ≥ 115% | 8% FYC | |

| % Actual P13 rolling of location | % P13 modifier |
|---|---|
| < 50% | x Actual P13 |
| ≥ 50% and < 55% | 75% |
| ≥ 55% and < 60% | 90% |
| ≥ 60% and < 65% | 100% |
| ≥ 65% and < 70% | 105% |
| ≥ 70% and < 75% | 110% |
| ≥ 75% | 115% |
| No P13 yet | 100% |
