# Curriculum Content Files — Quality Rewrite

## Background

The 20 curriculums were originally generated by `hugo/create_20.py` using the structure of the template curriculum **"Rewriting Extinction"** (`gA1W24Ga6lXwdHHx`). While the structural skeleton (4 sessions, activity types, vocab lists, readings) was correct, the **text quality** of several activity types was far below the template's standard.

## Quality Bar (from template: Rewriting Extinction)

### Preview
The template preview is a **rich, captivating paragraph** (~150 words) that:
- Opens with vivid, imaginative hooks ("What if the woolly mammoth could walk the tundra again?")
- Names the 10 vocabulary words and explains how they'll be learned
- Describes the learning journey and what the learner will be able to do by the end
- Reads like compelling marketing copy, not a dry description

**Bad (generated):** "Every night your brain embarks on an extraordinary journey of repair, memory consolidation, and emotional processing. Learn 10 words through cutting-edge sleep science."
**Good (template style):** A full paragraph with vivid hooks, word list, learning journey description, and a compelling closing line.

### introAudio (Session 1 & 2)
The template introAudio text is an **elaborate teaching script** (~500-800 words) that:
- Welcomes the learner and sets the scene with vivid context
- Lists all 5 vocabulary words for the session
- **Teaches each word individually**: part of speech, full definition, example sentence from the article context, explanation of how the word is used in the article
- Transitions smoothly between words with connecting phrases
- For Session 2: recaps Session 1 words before introducing new ones

**Bad (generated):** "In this session you'll learn: Circadian, Consolidation, Deprivation, Cognitive, Neurological."
**Good (template style):** A full teaching script that walks through each word with definitions, examples, and article context.

### introAudio (Session 3 — Review)
The template review intro:
- Congratulates the learner on learning all 10 words
- Briefly recaps what was covered in Sessions 1 and 2 with specific references
- Explains what the review session will involve
- Motivates the learner for the full article in Session 4

**Bad (generated):** "Review all 10 words before the full article in Session 4!"
**Good (template style):** A paragraph recapping all 10 words in context and explaining the review plan.

### introAudio (Session 4 — Final Reading Intro)
The template final intro:
- Welcomes to the final session
- Recaps the learning journey across all sessions
- Lists all 10 words the learner will encounter
- Sets expectations for the full article reading

**Bad (generated):** "Read the complete article about the invisible architecture of sleep. Notice how each word fits naturally."
**Good (template style):** A full paragraph setting up the final reading experience.

### introAudio (Session 4 — Farewell)
The template farewell is an **extensive vocabulary review** (~400-600 words) that:
- Reviews each of the 10 words with definition and a fresh example sentence
- Summarizes the learning journey
- Provides a warm, encouraging farewell

**Bad (generated):** "Congratulations on completing 'The Invisible Architecture of Sleep'! You mastered: Circadian, Consolidation, Deprivation, Cognitive, Neurological, Insomnia, Melatonin, Restorative, Chronic, Impairment. Well done!"
**Good (template style):** A full review of all 10 words with definitions and examples, plus a warm farewell.

### writingSentence items
The template writingSentence items have:
- Detailed prompts that specify the context for using the word
- An example sentence showing correct usage
- Format: "Use the word 'X' in a sentence about [specific context]. Example: [full example sentence]."

**Bad (generated):** "Use 'circadian' in a sentence about the invisible architecture of sleep."
**Good (template style):** "Use the word 'circadian' in a sentence about biological rhythms. Example: Every organism on Earth follows a circadian rhythm synchronized to the 24-hour cycle of light and darkness."

### writingParagraph prompts
The template writingParagraph prompts are:
- Specific to the topic and session content
- Guide the learner to use vocabulary in meaningful analytical writing
- Reference specific concepts from the article

**Bad (generated):** "Explain a key concept from the invisible architecture of sleep using Session 1 vocabulary."
**Good (template style):** "Explain how the brain uses circadian rhythms and neurological processes to perform cognitive consolidation during sleep. What happens when this process is disrupted by deprivation?"

## What Gets Stripped (mp3Url removal)
- `preview.mp3Url` — removed so audio can be regenerated
- `introAudio` activities — text is rewritten, audio will be regenerated
- `writingSentence` items `mp3Url` — removed from each item so audio can be regenerated
- `writingParagraph` `mp3Url` — removed so audio can be regenerated

## What Does NOT Change
- Activity structure (types, order, count)
- Vocabulary lists
- Reading passages (s1_reading, s2_reading, full_article)
- Activity types that aren't introAudio/writingSentence/writingParagraph (flashcards, vocab levels, reading, speakReading, readAlong)
- Description fields on non-rewritten activities

## File Structure

Each file in `hugo/curriculums/` is a Python file named after a slug of the curriculum title. Each file exports a dict `CONTENT` with:

```python
CONTENT = {
    "curriculum_id": "...",          # ID from the database
    "title": "...",                   # Curriculum title
    "preview": "...",                 # Rich preview text (no mp3Url)
    "s1_intro": "...",               # Session 1 introAudio text
    "s2_intro": "...",               # Session 2 introAudio text
    "s3_intro": "...",               # Session 3 review introAudio text
    "s4_intro": "...",               # Session 4 final reading introAudio text
    "s4_farewell": "...",            # Session 4 farewell introAudio text
    "s1_ws_items": [...],            # Session 1 writingSentence items (prompt + targetVocab, no mp3Url)
    "s2_ws_items": [...],            # Session 2 writingSentence items
    "s1_wp_prompts": [...],          # Session 1 writingParagraph prompts
    "s2_wp_prompts": [...],          # Session 2 writingParagraph prompts
    "s3_wp_prompts": [...],          # Session 3 writingParagraph prompts
    "s4_wp_prompts": [...],          # Session 4 writingParagraph prompts
}
```

## Update Script

`hugo/update_curriculums.py` reads each curriculum content file, fetches the existing curriculum from the API/DB, patches the relevant fields, and uploads the update.

## Methodology

1. Query the template curriculum (`gA1W24Ga6lXwdHHx`) from the database
2. Analyze every activity type and compare text quality between template and generated
3. Identify all fields that fall below the quality bar
4. For each of the 20 curriculums, write elaborate replacement text matching the template's depth
5. Strip `mp3Url` from `preview`, `writingSentence` items, and `writingParagraph` so audio can be regenerated
6. Upload updates via the curriculum API
