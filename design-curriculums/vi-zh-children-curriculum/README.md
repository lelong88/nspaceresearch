## Recreation Instructions

If any curriculum needs to be recreated:
1. Query the DB for the curriculum content: `SELECT content FROM curriculum WHERE id = '<id>';`
2. Use the content JSON as a template
3. Strip any generated keys (mp3Url, illustrationSet, etc.) before re-uploading
4. Re-create via `curriculum/create` with `language="zh"`, `userLanguage="vi"`
5. Re-add to the appropriate series, set display order and price
