## Recreation Instructions

1. **Infrastructure** (collections + series): Run `orchestrator.py`. Creates 5 collections and 25 series, wires them together, sets display orders. IDs logged to stdout.
2. **Curriculums**: The `create_*.py` scripts create individual curriculums. Run beginner scripts first, then preintermediate, then intermediate. Each script:
   - Creates the curriculum via API
   - Adds it to the appropriate series
   - Sets display order and price
3. **Validation**: Use `validate_content.py` to check content JSON before upload.
4. **API helpers**: Use `api_helpers.py` for all API calls.

> The orchestrator is NOT idempotent — do not re-run without deleting existing infrastructure first.
