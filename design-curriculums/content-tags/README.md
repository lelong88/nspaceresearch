# Content Type Tags

Custom tags added to `curriculum.content['curriculumTags']['tags']` for UI display purposes.

## Tags

| Tag | Description | Source |
|---|---|---|
| `movie` | Movie clip-based vocabulary curriculum | Has `youtubeUrl`, title contains "Phim" or "Film" |
| `music` | Song-based vocabulary curriculum | Has `youtubeUrl`, title contains "Bài Hát" or "Song" |
| `podcast` | Podcast/TED Talk-based vocabulary curriculum | Has `youtubeUrl`, title contains "Podcast" |
| `story` | Fiction story/novel chapter curriculum | In fiction series (see list below) |

## How Tags Are Stored

Tags are appended to the existing `curriculumTags.tags` array, which is auto-generated by the platform. Example:

```json
{
  "curriculumTags": {
    "tags": [
      "reader",
      "preintermediate", 
      "vocab_intermediate",
      "reading_preintermediate",
      "story"  // <-- custom tag added here
    ],
    "totalPracticeMinutes": 44,
    "practiceMinutesByType": { ... }
  }
}
```

## ⚠️ Known Issue

The platform's content generation pipeline may regenerate `curriculumTags` and overwrite custom tags. If tags disappear, re-run the tagging script.

## How to Re-Tag

### 1. Find curriculums with youtubeUrl

```sql
SELECT id, content->>'title' as title, content->>'youtubeUrl' as youtube_url
FROM curriculum 
WHERE content->>'youtubeUrl' IS NOT NULL
ORDER BY content->>'title';
```

Classify by title pattern:
- "Học Qua Bài Hát" / "Learn Through Song" → `music`
- "Học Qua Phim" / "Learn Through Film" → `movie`
- "Học Qua Podcast" → `podcast`

### 2. Find fiction story curriculums

Query curriculums in these fiction series:

| Series | ID | Novel | Language |
|---|---|---|---|
| The Garden of Forgotten Letters | `f5ort29f` | Coming-of-age/family drama | vi-en |
| The Last Light of Alder House | `70b5bb22` | Drama | vi-en |
| The Little Bookshop by the Sea | `n4y9zm3v` | Drama | vi-en |
| The Signal Beyond | `2wlmlwpz` | Sci-fi | en-en |
| The Silent Clocktower | `r8vwa6hv` | Mystery | vi-en |
| 味道的记忆 (Memories of Flavor) | `uq7ezeuh` | Drama | vi-zh |
| 消失的画 (The Vanishing Painting) | `wlzqfag8` | Mystery | vi-zh |
| 湖边的琴声 (Sound of Music by the Lake) | `z6xddztr` | Romance | vi-zh |
| Memories of Flavor (en-zh mirror) | query by title | Drama | en-zh |
| The Vanishing Painting (en-zh mirror) | query by title | Mystery | en-zh |
| The Sound of Music by the Lake (en-zh mirror) | query by title | Romance | en-zh |

```sql
-- Get all curriculums in fiction series
SELECT c.id, c.content->>'title' as title
FROM curriculum c
JOIN curriculum_series_items csi ON c.id = csi.curriculum_id
WHERE csi.curriculum_series_id IN (
  'f5ort29f', '70b5bb22', 'n4y9zm3v', '2wlmlwpz', 'r8vwa6hv',
  'uq7ezeuh', 'wlzqfag8', 'z6xddztr'
)
ORDER BY csi.curriculum_series_id, c.display_order;

-- Get en-zh mirror novels (not in series, query by title pattern)
SELECT id, content->>'title' as title
FROM curriculum
WHERE language = 'zh' AND user_language = 'en'
AND (content->>'title' LIKE '%Memories of Flavor%'
     OR content->>'title' LIKE '%Vanishing Painting%'
     OR content->>'title' LIKE '%Sound of Music by the Lake%');
```

### 3. Add tags via API

For each curriculum:

1. Fetch content via `curriculum/getOne`
2. Check if tag already exists in `content['curriculumTags']['tags']`
3. If not, append the tag to the array
4. Update via `curriculum/update` with the modified content

```python
import json
import requests
from firebase_token import get_firebase_id_token

API_BASE = "https://helloapi.step.is"
UID = "zs5AMpVfqkcfDf8CJ9qrXdH58d73"

def add_tag(curriculum_id: str, new_tag: str):
    token = get_firebase_id_token(UID)
    
    # Fetch
    resp = requests.post(f"{API_BASE}/curriculum/getOne",
        json={"firebaseIdToken": token, "id": curriculum_id})
    content = resp.json().get("content", {})
    if isinstance(content, str):
        content = json.loads(content)
    
    # Check/add tag
    tags = content.get("curriculumTags", {}).get("tags", [])
    if new_tag in tags:
        return False  # already exists
    
    tags.append(new_tag)
    content["curriculumTags"]["tags"] = tags
    
    # Update
    requests.post(f"{API_BASE}/curriculum/update",
        json={"firebaseIdToken": token, "id": curriculum_id, "content": content})
    return True
```

### 4. Verify

```sql
SELECT id, content->>'title' as title, content->'curriculumTags'->'tags' as tags
FROM curriculum 
WHERE content->'curriculumTags'->'tags' ? 'movie'
   OR content->'curriculumTags'->'tags' ? 'music'
   OR content->'curriculumTags'->'tags' ? 'podcast'
   OR content->'curriculumTags'->'tags' ? 'story'
LIMIT 20;
```
