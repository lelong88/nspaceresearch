## How Content Was Created

Each curriculum was created by a standalone Python script with all text content hand-written inline. Scripts implemented `build_content()`, `validate()`, and `create()` functions. The `validate()` function checked all 3 correctness properties (content structure vs CONTENT_CORRUPTION_RULES, lowercase vocabList, no strip-keys) before any API call. After successful creation and verification, all scripts were deleted.
