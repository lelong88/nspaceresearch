# Tiếng Nhật Cho Người Mới Bắt Đầu (Vietnamese-Japanese Beginner Curriculums)

20 Japanese-learning curriculums for Vietnamese-speaking adults at the beginner level, organized into 1 collection and 3 series.

- **Language pair**: `userLanguage="vi"`, `language="ja"` (Vietnamese speakers learning Japanese)
- **Audience**: Vietnamese adults beginning their Japanese learning journey
- **Method**: Python scripts via `curriculum/create` API (scripts deleted after creation)
- **Script directory**: `vi-ja-beginner-curriculums/`
- **Vocabulary writing system**: Hiragana and katakana only (no kanji); romaji used in introAudio for pronunciation guidance

## Recreation Instructions

Source materials (per-curriculum `create_*.py` scripts, `orchestrator.py`, `validate_content.py`) are deleted after successful import to the database. Full content for any curriculum is recoverable from the DB via `curriculum/getOne` or MCP postgres queries on the `curriculum` table.

Original creation method (documented in `.kiro/specs/vi-ja-beginner-curriculums/`):

- One standalone Python script per curriculum, hand-crafted Japanese (hiragana/katakana, no kanji) and Vietnamese marketing text
- One orchestrator script (`orchestrator.py`) created the collection and 3 series and wired them together
- A beginner-specific `validate_content.py` validated each content JSON before upload (NO lowercase enforcement for Japanese vocabulary; format-aware checks for `beginner_mini` vs `beginner_short`)
- Shared root-level `api_helpers.py` handled all API calls
- Auth via `firebase_token.get_firebase_id_token('zs5AMpVfqkcfDf8CJ9qrXdH58d73')`
- API base: `https://helloapi.step.is`

To recreate any curriculum:

1. **Retrieve content from DB** — `curriculum/getOne` API or `SELECT content FROM curriculum WHERE id = '<id>'`
2. **Strip generated keys** — remove `mp3Url`, `illustrationSet`, `chapterBookmarks`, `segments`, `whiteboardItems`, `userReadingId`, `lessonUniqueId`, `curriculumTags`, `taskId`, `imageId` from the content JSON before reuploading
3. **Create new curriculum** — `curriculum/create` with top-level body fields `language="ja"`, `userLanguage="vi"`, and `content` (JSON string)
4. **Add to series** — `curriculum-series/addCurriculum` (or `curriculum-collection/addCurriculumToSeries`) with the appropriate series ID
5. **Set display order** — `curriculum/setDisplayOrder` with the curriculum ID and order from the table above
6. **Set price** — `curriculum/setPrice` with `price=9` (mini) or `price=19` (short)
7. **Replace existing** — to swap a curriculum, first `curriculum-series/removeCurriculum`, then `curriculum/delete`, then create the replacement
8. **Do NOT call `curriculum/setPublic`** — these curriculums remain private until content generation (audio, illustrations) completes
