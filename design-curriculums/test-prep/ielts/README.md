# Luyện Thi IELTS: Từ Vựng Học Thuật

- Status: in-progress (scripts not yet imported)
- Language: vi-en, Level: intermediate
- Target: Vietnamese learners aiming for IELTS 4.5–5.5
- Structure: 8 curriculums × 18 words = 144 words
- Sessions: 5 per curriculum (3 learning + review + full reading)
- Collection: Học Từ Vựng Theo Chủ Đề (`279d6843`)

## How to create
1. Run `create_ielts_vi_series.py` to create the series and add it to the collection
2. Run individual curriculum scripts (`ielts_vi_curriculum_1.py`, etc.) to create each curriculum
3. After successful import, delete scripts and update this README with series ID and curriculum table
