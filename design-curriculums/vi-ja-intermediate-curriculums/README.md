# Tiếng Nhật Nâng Cao (Vietnamese-Japanese Intermediate / Upper-Intermediate Curriculums)

20 Japanese-learning curriculums for Vietnamese-speaking adults at the intermediate and upper-intermediate levels, organized into 1 collection and 5 series of 4 curriculums each.

- **Language pair**: `userLanguage="vi"`, `language="ja"` (Vietnamese speakers learning Japanese)
- **Audience**: Vietnamese adults expanding from beginner Japanese into adult-topic vocabulary across business, culture, society, science, and personal development
- **Method**: Python scripts via `curriculum/create` API (scripts deleted after creation)
- **Script directory**: `vi-ja-intermediate-curriculums/`
- **Vocabulary writing system**: Kanji with selective furigana — N3 kanji freely + N2+ furigana for the intermediate format; N3-N2 freely + rare-kanji furigana for the upper-intermediate format
- **Pricing**: 49 credits per curriculum (all 20)
- **Visibility**: All curriculums remain **private** (`isPublic=false`) until content generation (audio, illustrations) completes

## Format Specifications

Both formats share the same activity placement (vocabLevel3 in Session 4, writingParagraph in Session 5) and the same 5-session shape with 18 words. They differ in content complexity (passage length, analytical depth, furigana density).

| Field | intermediate | upper_intermediate |
|---|---|---|
| Sessions | 5 | 5 |
| Vocabulary | 18 words (3 groups of 6) | 18 words (3 groups of 6) |
| vocabLevel3 | Required in Session 4 | Required in Session 4 |
| writingParagraph | Required in Session 5 | Required in Session 5 |
| Reading length (S1-S3) | 250-400 chars | 350-500 chars |
| Reading length (S5 final) | 600-900 chars | 800-1200 chars |
| Kanji policy | N3 freely + N2+ furigana | N3-N2 freely + rare-kanji furigana |
| Price | 49 | 49 |
| Count | 10 (curriculums #1-10) | 10 (curriculums #11-20) |

Per-session activity outline (both formats):

- **Sessions 1-3** — `introAudio`, `viewFlashcards`, `speakFlashcards`, `vocabLevel1`, `vocabLevel2`, `introAudio` (grammar), `reading`, `speakReading`, `readAlong`, `writingSentence` ×3
- **Session 4** — review with all 18 words: `vocabLevel1`, `vocabLevel2`, `vocabLevel3`, `writingSentence` ×4-5
- **Session 5** — final reading + `writingParagraph` (≥6 vocab words, ≥2 guiding prompts) + farewell `introAudio`

`contentTypeTags = []` for all 20.
