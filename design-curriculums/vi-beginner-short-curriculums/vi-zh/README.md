## SQL Verification Queries

```sql
-- Count all vi-zh curriculums in this collection (expect 30)
SELECT COUNT(*)
FROM curriculum c
JOIN curriculum_series_items csi ON c.id = csi.curriculum_id
JOIN curriculum_collection_series ccs ON csi.curriculum_series_id = ccs.curriculum_series_id
WHERE ccs.curriculum_collection_id = 'xjabf53e'
AND c.uid NOT LIKE '%_deleted';

-- Language homogeneity check (all should be language=zh, user_language=vi)
SELECT c.language, c.user_language, COUNT(*)
FROM curriculum c
JOIN curriculum_series_items csi ON c.id = csi.curriculum_id
JOIN curriculum_collection_series ccs ON csi.curriculum_series_id = ccs.curriculum_series_id
WHERE ccs.curriculum_collection_id = 'xjabf53e'
AND c.uid NOT LIKE '%_deleted'
GROUP BY c.language, c.user_language;

-- Price check (all should be price=9)
SELECT c.price, COUNT(*)
FROM curriculum c
JOIN curriculum_series_items csi ON c.id = csi.curriculum_id
JOIN curriculum_collection_series ccs ON csi.curriculum_series_id = ccs.curriculum_series_id
WHERE ccs.curriculum_collection_id = 'xjabf53e'
AND c.uid NOT LIKE '%_deleted'
GROUP BY c.price;

-- Duplicate title check
SELECT c.content->>'title' as title, COUNT(*) as cnt
FROM curriculum c
WHERE c.uid = 'zs5AMpVfqkcfDf8CJ9qrXdH58d73'
AND c.uid NOT LIKE '%_deleted'
AND c.user_language = 'vi'
AND c.language = 'zh'
GROUP BY c.content->>'title'
HAVING COUNT(*) > 1;

-- List all curriculums by series with display orders
SELECT cs.title as series_title, c.id, c.content->>'title' as title, c.display_order, c.price
FROM curriculum c
JOIN curriculum_series_items csi ON c.id = csi.curriculum_id
JOIN curriculum_series cs ON csi.curriculum_series_id = cs.id
JOIN curriculum_collection_series ccs ON csi.curriculum_series_id = ccs.curriculum_series_id
WHERE ccs.curriculum_collection_id = 'xjabf53e'
AND c.uid NOT LIKE '%_deleted'
ORDER BY cs.display_order, c.display_order;
```

## Recreation

Each curriculum has 1 session with activities following one of 4 templates:

- **balanced_skills**: introAudio → viewFlashcards → speakFlashcards → vocabLevel1 → vocabLevel2 → reading → speakReading → readAlong → writingSentence → introAudio (farewell)
- **speaking_focus**: introAudio → viewFlashcards → speakFlashcards → reading → speakReading → readAlong → introAudio (farewell)
- **reader**: introAudio → viewFlashcards → speakFlashcards → reading → readAlong → introAudio (farewell)
- **writing_lover**: introAudio → viewFlashcards → reading → readAlong → writingSentence → writingParagraph → introAudio (farewell)

Vocabulary per curriculum: 3-5 words (simplified Chinese characters). All user-facing text in Vietnamese, reading passages in Chinese.
All curriculums are private (`is_public = false`), priced at 9 credits.

Source scripts were in this directory (deleted after successful import). Content is fully recoverable via `curriculum/getOne` with each curriculum ID.
