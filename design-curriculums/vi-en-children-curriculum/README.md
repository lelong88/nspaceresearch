## Recreation Instructions

If curriculums need to be recreated:

1. **Retrieve content from DB** — use `curriculum/getOne` API or MCP postgres to get the full content JSON for any curriculum by ID
2. **Create new curriculum** — call `curriculum/create` with `language="en"`, `userLanguage="vi"`, and the content JSON
3. **Add to series** — call `curriculum-series/addCurriculum` with the series ID and new curriculum ID
4. **Set display order** — call `curriculum/setDisplayOrder` with the curriculum ID and correct order number
5. **Set price** — call `curriculum/setPrice` with the curriculum ID and correct price (9/19/49)
6. **Remove old** — if replacing, remove the old curriculum from the series first via `curriculum-series/removeCurriculum`, then delete via `curriculum/delete`
7. **Do NOT call `setPublic`** — all children's curriculums remain private until content generation is complete
