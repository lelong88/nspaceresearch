## Recreation Instructions

1. **Infrastructure** (collections + series): Run `orchestrator.py`. Creates 5 collections and 25 series, wires them together, sets display orders. IDs logged to stdout.
2. **Update scripts**: After running the orchestrator, replace all `PLACEHOLDER_{series}` values in the `create_*.py` scripts with the real series IDs from the orchestrator output.
3. **Curriculums**: Run the `create_*.py` scripts in order — beginner first, then preintermediate, then intermediate. Each script:
   - Creates the curriculum via API (`language="fr"`, `userLanguage="vi"`)
   - Adds it to the appropriate series
   - Sets display order and price (beginner=19, standard=49)
4. **Validation**: Use `validate_content.py` to check content JSON before upload.
5. **API helpers**: Use `api_helpers.py` for all API calls.
6. **Session titles**: Vietnamese — Phần 1, Phần 2, Phần 3, Ôn tập, Đọc tổng hợp (beginner uses only Phần 1, Phần 2, Ôn tập, Đọc tổng hợp).

> The orchestrator is NOT idempotent — do not re-run without deleting existing infrastructure first.
