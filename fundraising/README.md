# PRODUCT
Step is an AI-powered language learning mobile app built around interesting contents that users actually enjoy. Users learn through structured curriculums containing sessions with diverse activity types: reading (ebooks, read-along), speaking (pronunciation, roleplay), writing (sentences, paragraphs), vocabulary (flashcards), and listening exercises.

- Target market: People who want to, need to learn a language but don't have enough self-discipline and/or time to do it. They learn language with Step while doing something else: read a fiction story, read news, learn a new interesting concept, etc.

# WHAT I NEED
## A full-time employee 
- Title: Learning Content Manager
- Expected salaray gross pay: $3000/month (+ ESOP benefits)
- Qualifications: IELTS 8.0+ or equivalent, TESOL is optional but recommended, experience in user research, 5+ years experience in language teaching, can also work independently and creatively.
- Why: The curriculum store is created and managed by AI agents based on human-input deep understanding of diverse learning needs. Each learning need discovered and templated for (a human-designed template curriculum) can be replicated instantly to create years of learning materials for this need and all learners in it. We are bottle-necked at exactly this iteratic human-driven process:

  1. discover a learning need
  2. write a template curriculum for this need
  3. test the template curriculum in real world use
  4. finetune the template further
  5. test
  6. finetune
  7. ... until the template is of highest possible quality before we let the AI agent replicate into 1000s of derivative curriculums. 
  
This process is at the core of the product and offers very high leverage because this 1 template done well will be equivalent with 1000s of derivative curriculums done well, and vice versa. The same process applies even when we work with partners such as language center schools, individual teachers, etc. The same process applies for all target languages (English, Chinese, Spanish, German, Hindi, etc.) across all markets
